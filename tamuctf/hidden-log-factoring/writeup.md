## 2.Attack strategy 

### 2.1 Basic Idea
- DLP components of the problem hint many 
    - given modulus p 
    - small secret     
$$
s < 2^{100} 
$$

- We can try to solve discrete log problem to find secret s 
- If we can find s , we can recorver secret key d of RSA 
- Then we can solve factorization problem of modulus n 

=> Find square root of ciphertext => achieve flag 

### 2.1 quan sát p 
- We can try factorize ord of group is p-1 with limit for factors to observe
```python 
from sympy import factorint

def factorize_p1(p, limit=10**7):
    p1 = p - 1
    factors = factorint(p1)
    
    small_factors = {k: v for k, v in factors.items() if k < limit}
    
    remaining = p1
    for k, v in small_factors.items():
        remaining //= (k ** v)
    
    return small_factors, remaining
```

### 2,3 Find secret 
- Cause $s < 2^{101} => s \pmod{2^{101}} = s$ 

=>  We can put this problem into discrete logarithm in subgroup order $2^{101}$ to find $s = s \bmod{2^{101}}$ 

- Especially with order of subgroup is exponential of 2 
    - => We can find each bit of $s$  in order
    - => Or find step by step $s \bmod{2}$ , $s \bmod{2^{2}}$ , ......., $s \bmod{2^{101}}$

```python 
def solve_dlp_binary_subgroup(A, g, p, num_bits):
    s = 0
    current_A = A
    m = (p - 1) // (2**num_bits)
    g_base = pow(g, m, p)
    A_base = pow(A, m, p)

  
    g_check = pow(g_base, 2**(num_bits - 1), p)
    
    for i in range(num_bits):
        exponent = 2**(num_bits - 1 - i)
        bit_tester = pow(A_base, exponent, p)
        
        if bit_tester == g_check:
            bit = 1
        else:
            bit = 0
            
        s += bit * (2**i)
        
        if bit == 1:
            inv_g = pow(g_base, - (2**i), p)
            A_base = (A_base * inv_g) % p
            
    return s
```

- We find $s = 485391067385099231898174017598$ 

### 2,4 Factorize n 
- With $s$ , we can recover $d$ by given function ***hkdf_mask(s)***
- After find $d$ , we see that :
$$
d\times e -1 = k\phi(n) 
$$
$$
=>\frac {d\times e -1}{n} \approx k 
$$
$$
=> \phi(n) = \frac {d\times e -1}{k}
$$

- Then we can find $p,q$ from $\phi(n)$ 

### 2,5 Recover flag

- When we have p,q , we can find square root modulo n of c 
    - Find square root modulo p and modulo q 
    - Use ***Chinese remainder theorem*** to combine them
    - Check which in 4 solutions is flag

def hkdf_mask(secret: bytes, length: int) -> bytes:
    hkdf = HKDF(
        algorithm=hashes.SHA256(),
        length=length,
        salt=None,
        info=b"rsa-d-mask",
        backend=default_backend()
    )
    return hkdf.derive(secret)

# factorize n 
def factor_with_d_algebraic(n, e, d):
    
    k_est = (e * d) // n
    
    for k in range(k_est, k_est + 3):
        if k == 0:
            continue
        if (e * d - 1) % k != 0:
            continue
            
        phi_n = (e * d - 1) // k
        S = n - phi_n + 1
        delta = S * S - 4 * n
        if delta < 0:
            continue
        sq_delta = isqrt(delta)
        if sq_delta * sq_delta == delta: 
            q1 = (S + sq_delta) // 2
            q2 = (S - sq_delta) // 2      
            if q1 * q2 == n:
                return q1, q2           
    return None
